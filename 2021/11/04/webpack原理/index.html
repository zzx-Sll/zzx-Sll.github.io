webpack打包的原理

手写一个简单的webapck

  • 准备工作：
    • 新建一个项目名文件夹
    • npm init -y 初始化一个项目
    • 根目录下建立一个 bin 文件夹
    • 在bin目录下建立一个打包的主程序js文件（文件名自定义）
    • 在主程序文件下顶部对该程序文件进行声明当前的运行环境为node #!/usr/bin/env node
    • 在package.json文件下进行bin节点的声明，建立自己的打包脚本指令，指定打包的文件路径。
    • npm link 将自己打包程序文件链接到全局
    • 然后执行打包指令就可以在node环境下输出了
