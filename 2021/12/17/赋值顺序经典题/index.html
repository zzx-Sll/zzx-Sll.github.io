赋值顺序经典题

发表于 2021-12-17 分类于 前端
  • 一个经典的面试题
  • 输出: undefined {n:2}
  • 解析:
    • 知识点前提: 赋值操作顺序是从右到左,b=a是浅拷贝。a,b指向的是堆上的同一个地址(同一个对象)
      • ```javascript
        var a=1,b=2,c=3;
        a=b=c
        console.log(a,b,c) // ouput 3,3,3
    • 执行 a.x操作
      • ```javascript
        a.x //此时a 为旧对象 {n:1}。注意:此时a被解析为旧对象！！！
        // 所以这步执行后结果为 a={n:1,x:undefined} b={n:1,x:undefined}
    • 执行 a.x=a
    • 最后执行 console.log(a.x) console.log(b.x),此时又会重新解析a,b 而a={n:2},访问对象上不存在的属性输出undefined,而此时b的值为{n:1,x:{n:2}},所以访问b.x 输出 {n:2}
